**🧱 Brick Breaker Applicati**on

A simple Brick Breaker game built using Java (JDK 21) following a clean package structure with model and service layers.

**📌 Project Overview**

This project is a classic Brick Breaker arcade game where:

Player controls a paddle

Ball bounces and breaks bricks

Score increases as bricks are destroyed

Game ends when ball falls below paddle
The project is structured using proper separation of concerns:

main → Entry point

model → Game objects (Ball, GameBoard)

service → Game logic handling

**🏗️ Project Structure**

BrickBreakerApplication
│
├── com.brickbreaker.main
│   └── Main.java
│
├── com.brickbreaker.model
│   ├── Ball.java
│   └── GameBoard.java
│
├── com.brickbreaker.service
│   └── GameService.java
│
└── module-info.java

**⚙️ Technologies Used**

Java 21

Object-Oriented Programming (OOP)

Modular Project Structure

**🎮 Game Features**

Ball movement with collision detection

Paddle control

Brick breaking logic

Score tracking

Game over condition
